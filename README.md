Overview: This application is a simple desktop expense tracker built
with Python using the Tkinter library. It allows users to record,
manage, and analyze their personal expenses in a convenient graphical
interface.

Main Idea: The main idea of the application is to help users keep track
of their spending habits, understand where their money goes, and make
better financial decisions. It provides basic tools for adding, editing,
deleting, and viewing expenses, as well as simple analytics through
charts.

Key Features: - Add expenses with name, category, amount, and automatic
date - Edit and delete existing expenses - Filter expenses by date
range - Search expenses by name or category - Sort expenses by date,
amount, or category - Visualize expenses using charts (by category and
by date) - Automatic calculation of total spending - Highlight large
expenses with a warning - Data encryption for basic privacy protection

How It Works: All expenses are stored in an encrypted file. The
application uses a simple XOR-based encryption combined with base64
encoding to protect data from being easily read.

The graphical interface is built with Tkinter and includes a table for
displaying expenses and buttons for interacting with the data.

Purpose: This project is designed for learning purposes and
demonstrates: - GUI development in Python - File handling and data
storage - Basic encryption techniques - Data visualization with
matplotlib

Conclusion: The Expense Tracker is a useful beginner-friendly project
that combines multiple programming concepts into a practical tool for
everyday use.
